# Whispy

Whispy is a speech to text app.

## Running


Make a virtual environment
```
python3 -m venv .venv
```

Activate virtual environment

Windows:
```
.venv\Scripts\activate
```

Linux/macOS:
```
source .venv/bin/activate
```

```
uv sync
uv python app.py
```

And that's it! You can now use a Whispy
